Welcome to the PICO W Water System BLE Panel! This interface allows you to control and monitor your Raspberry Pi Pico-based watering system via Bluetooth Low Energy (BLE).

ğŸ”— Main Control

Use this tab to connect to your Pico device, read system logs, and control the relay manually.

  • Scan & Connect: Scan for and connect to nearby Pico devices.
  • Read Log File: Retrieve the current log from the device.
  • Save Log: Save the collected log data to your computer.
  • Clear Log: Erase the log on the device (irreversible).
  • Set Device Time: Set the date and time on the device.
  • Manual Relay ON/OFF: Manually turn the relay on or off.

ğŸ“¶ WiFi Control

Manage the device's WiFi connectivity and file operations.

  • Turn WiFi ON/OFF: Enable or disable WiFi on the device.
  • WiFi Status: Check the current WiFi connection status.
  • List Files: View files stored on the device.
  • Download/Upload/Delete Files: Transfer files to/from the device.

Note: WiFi file operations use HTTP to connect to the Pico's web server at 192.168.4.1:5001. Make sure WiFi is ON and you are connected to the Pico's WiFi network.

ğŸ›  Schedule

Create and send watering schedules to the device.

  • Generate Schedule: Create a recurring schedule based on start time, duration, interval, and count.
  • Batch Schedule Generator: Manually enter multiple schedule entries.
  • Send Schedule File: Upload the schedule to the device.
  • Read Schedule: Retrieve the current schedule from the device.
  • Clear Schedule: Remove all scheduled events from the device.

ğŸ”§ Manual Control

Direct control over device functions.

  • Reboot Device: Restart the Pico device.
  • Relay Control: Turn the water relay on or off manually.
  • Set RTC Time: Manually set the device's real-time clock to a specific date and time.
  • Sync with PC Time: Automatically synchronize the device's clock with your computer's current time.
